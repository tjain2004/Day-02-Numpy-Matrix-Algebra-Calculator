# Day-02-Numpy-Matrix-Algebra-Calculator
What  Matrix Algebra System (MAS) Will Do

This calculator will support:

1. Matrix Creation

   a)Enter any matrix manually

   b)It will store it as a NumPy array

2. Basic Matrix Operations

   a)Matrix Addition

   b)Matrix Subtraction

   c)Matrix Multiplication

   d)Scalar Multiplication

3. Advanced Linear Algebra Operations

   a)Matrix Transpose

   b)Determinant

   c)Rank

   d)Inverse

   e)Eigenvalues

   f)Eigenvectors

4. System of Linear Equations Solver

   Solves equations like
   Ax = b
